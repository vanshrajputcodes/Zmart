# 🛒 Zmart — Smart Shopping Platform

Zmart is a modern, production-ready e-commerce web application built specifically for the Indian market. It focuses on providing a seamless shopping experience with smart recommendations, UPI-based payments, and a powerful admin dashboard.

---

## 🚀 Project Overview

Zmart solves real-world problems faced by Indian online shoppers and small retailers:

- Lack of INR-native platforms with UPI integration
- High cart abandonment due to poor UX
- No intelligent product recommendation systems
- Complex admin tools requiring technical knowledge

Zmart addresses these challenges by combining performance, usability, and smart automation into one unified platform.

---

## ✨ Key Features

### 🛍️ Customer Features
- Product browsing with search & filters
- Smart product recommendations (AI-based)
- Wishlist and reviews system
- Smooth cart experience with real-time updates
- UPI-based checkout system
- Order tracking and history

### ⚙️ Admin Features
- Product management (CRUD with variants)
- Order and customer management
- Coupon and discount system
- Real-time analytics dashboard

### 🤖 Smart Recommendation Engine
- Collaborative filtering (user behavior)
- Content-based filtering (product attributes)
- Popularity fallback (for new users)

## 🧠 Tech Stack

| Layer        | Technology                          |
|-------------|-------------------------------------|
| Frontend     | React 18 + TypeScript               |
| Build Tool   | Vite                               |
| Styling      | Tailwind CSS + shadcn/ui           |
| Animation    | Framer Motion                      |
| State Mgmt   | Zustand + React Query              |
| Backend      | Supabase (PostgreSQL + Auth + RLS) |
| Charts       | Recharts                           |
| Validation   | Zod + React Hook Form              |

---

## 🏗️ System Architecture

- **Frontend:** React + Vite (fast & responsive UI)
- **State Management:** Zustand + React Query
- **Backend Services:** Supabase (Auth, DB, Storage)
- **Security:** Row-Level Security (RLS), JWT authentication
- **Infrastructure:** Cloud-based with CDN & auto-scaling
